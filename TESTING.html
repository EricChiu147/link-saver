🔖 Link Saver Extension - Testing Guide

This guide will help you test the Link Saver Chrome extension.

📋 Installation Steps

Step 1: Open Chrome and navigate to chrome://extensions/
Step 2: Enable "Developer mode" using the toggle in the top right corner
Step 3: Click "Load unpacked" button
Step 4: Select the extension directory (the folder containing manifest.json)
✅ Success: The extension should now appear in your extensions list and toolbar!

⚙️ Configuration

Step 1: Click the Link Saver extension icon in your toolbar
Step 2: Click the ⚙️ (Settings) button
Step 3: Enter your OpenAI API key (get one from OpenAI Platform)
Step 4: Click "Save" and optionally "Test Connection" to verify

🧪 Testing Features

1. Save a Link

Action: Navigate to any webpage (like this one!) and click the "💾 Save" button in the extension popup
Expected: An alert should confirm the link was saved, and it should appear in the links list

2. View Saved Links

Action: Click the extension icon to open the popup
Expected: You should see all your saved links with titles, URLs, and AI-generated summaries

3. AI Search

Action: In the popup, type a question in the "AI Search" field (e.g., "Show me links about testing")
Expected: The AI should analyze your links and provide relevant recommendations

4. Delete a Link

Action: Click the "🗑️ Delete" button next to any saved link
Expected: After confirming, the link should be removed from the list

🔍 Troubleshooting

Issue: "No API key configured" message
Solution: Go to Settings (⚙️) and configure your OpenAI API key
Issue: Extension not appearing in toolbar
Solution: Click the puzzle icon in Chrome toolbar and pin the Link Saver extension
Issue: Cannot save internal pages
Solution: Chrome extensions cannot access chrome:// or edge:// pages. Try saving a regular website instead.
