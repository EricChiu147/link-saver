⚙️ Settings

🤖 OpenAI API Configuration

Configure your OpenAI API key to enable automatic link summarization and AI-powered search. You can get your API key from OpenAI Platform.

ℹ️ About

Link Saver is a Chrome extension that helps you save and organize web links with AI-powered features.

  • 📌 Save links with one click
  • 🤖 Automatic AI summarization of saved pages
  • 🔍 AI-powered search through your saved links
  • 💾 All data stored locally using IndexedDB

📖 How to Use

  1. Configure your OpenAI API key above
  2. Click the extension icon to save the current page
  3. The extension will automatically generate a summary
  4. View all saved links by clicking the extension icon again
  5. Use the AI search to find relevant links by asking questions
